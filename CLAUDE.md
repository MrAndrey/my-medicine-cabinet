# Моя аптечка — CLAUDE.md

## Что это за проект
PWA-приложение для управления домашней аптечкой.
Основной сценарий: пользователь на телефоне быстро находит 
лекарство, видит где оно лежит и сколько осталось.
Пользователь: один человек, не технарь, основное устройство — 
телефон. Язык интерфейса: русский.

## Стек
- React + Tailwind
- localStorage для хранения данных
- PWA (работает на телефоне как приложение)
- Vercel Serverless Functions для проксирования API

## Деплой
- GitHub репо → Vercel автодеплой из ветки main
- URL: my-medicine-cabinet.vercel.app
- ANTHROPIC_API_KEY задан в Vercel Environment Variables

## Структура данных
Лекарство: {id, name, quantity, unit, location, expiry_date, category, notes}
Настройки: {expiry_warning_days: 7}
Места хранения: ["Шкаф в ванной", "Комод", "Аптечка на кухне"]
Список покупок: [{name}]

## Правила
- Крупные кнопки min 44px — приложение для телефона
- Адаптивность под мобильный обязательна
- Даты через dayjs, формат YYYY-MM-DD
- id через Date.now()
- Все строки в translations.js (подготовка к i18n)
- Кросс-браузерность обязательна: Safari/iPhone — основная платформа
