
Создай PWA-приложение «Моя аптечка» на React + Tailwind.

**1. Что это:**
Одностраничное PWA-приложение для управления домашней аптечкой — находить, добавлять и контролировать лекарства по месту хранения, количеству и сроку годности.

**2. Сценарий пользователя:**
1. Открыл — видит список лекарств, вверху бейдж "⚠️ N лекарств требуют внимания"
2. Нашёл нужное через поиск по названию или категории
3. Добавил новое — 4 обязательных поля за 20 секунд
4. Тапнул карточку — открыл детали, нажал −1 после приёма
5. Удалил закончившееся — перенёс в список покупок

**3. Экраны:**

Экран 1 — Список: бейдж с проблемными лекарствами, поиск, карточки (название, место, количество, срок), кнопка "Добавить". Карточки: белый/жёлтый/красный по статусу.

Экран 2 — Детали лекарства: все поля, кнопки −1/+1, "Редактировать", "Удалить". При удалении диалог "Добавить в список покупок?" Да/Нет.

Экран 3 — Форма добавления/редактирования: название (обязательно), количество + единица из списка таблетки/мл/упаковки (обязательно), место хранения — быстрый выбор из сохранённых + "добавить новое" (обязательно), срок годности (обязательно), категория из списка + своя (необязательно), заметки (необязательно).

Экран 4 — Список покупок: простой список, кнопка "Куплено" убирает позицию.

**4. Логика:**
- Срок < 7 дней → карточка жёлтая
- Срок прошёл → карточка красная
- Есть проблемные → показать бейдж
- Тапнул бейдж → фильтр только проблемных
- −1 → уменьшить количество, минимум 0
- Количество = 0 → предложить удалить или добавить в покупки
- Поиск → фильтр по названию и категории одновременно
- Новое место → сохранить в список мест
- Порог предупреждения — настраивается, по умолчанию 7 дней

**5. Данные:**
localStorage.
Лекарство: `{id, name, quantity, unit, location, expiry_date, category, notes}`
Настройки: `{expiry_warning_days: 7}`
Места хранения: `["Шкаф в ванной", "Комод", "Аптечка на кухне"]`
Список покупок: `[{name}]`

**6. Внешний вид:**
Светлый фон, зелёные и бежевые акценты. Крупные кнопки, адаптивно под мобильный. Цветовая индикация карточек: белый/жёлтый/красный.
