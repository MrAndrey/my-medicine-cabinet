# Changelog

## [0.1.0] — 2026-03-30

### Добавлено
- Полная реализация PWA «Моя аптечка» на React + Tailwind + Vite
- Экран списка: карточки лекарств с цветовой индикацией (белый/жёлтый/красный), поиск, бейдж с проблемными лекарствами, настройка порога предупреждения
- Экран деталей: просмотр всех полей, кнопки ±1, диалог удаления с предложением добавить в список покупок
- Форма добавления/редактирования: валидация обязательных полей, выбор места из сохранённых + добавление нового
- Экран списка покупок: добавление при удалении или нулевом остатке, кнопка «Куплено»
- localStorage для всех данных: лекарства, места хранения, список покупок, настройки
- PWA: manifest.json, service worker с cache-first стратегией, apple-touch-icon для Safari
- Иконки icon-192.png и icon-512.png (зелёный placeholder)
- Все строки интерфейса вынесены в translations.js (подготовка к i18n)

### Поле срока годности
- Заменён стандартный `<input type="date">` на два выпадающих списка: месяц и год
- Годы: 2020 — текущий+10
- Срок годности сделан необязательным: без даты карточка белая, в предупреждения не попадает
- Дата хранится как последний день выбранного месяца в формате YYYY-MM-DD (dayjs)

### Единицы измерения
- Добавлена единица «флакон»
- Добавлена возможность ввода своей единицы («Своя...»)
- При выборе «упаковки» появляется дополнительное поле «по N шт. в упаковке»

### Категории
- Пользовательские категории сохраняются в localStorage (ключ `customCategories`)
- При следующем открытии формы появляются в выпадающем списке наряду со стандартными

### Деплой и совместимость
- Исправлены пути к manifest.json, sw.js и иконкам для работы вне корня домена
- `base: '/'` в vite.config.js — корректная сборка для Vercel
- Service worker: добавлены `skipWaiting()` и `clients.claim()` для корректной активации в Safari
- Добавлен `<link rel="apple-touch-icon">` для установки на домашний экран iOS

### Исправлено
- 404 для JS/CSS на Vercel из-за `base: './'` и лишнего vercel.json с `rewrites`
- 404 для иконок PWA (созданы placeholder PNG)
- Safari отклонял файлы с неверным MIME типом — убран vercel.json, Vercel раздаёт статику из dist самостоятельно
